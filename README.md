# Introduction 
Example project for the GSBF.

Het Example Project is een TwinCAT 3 referentieproject dat laat zien hoe een compacte machine‑architectuur is opgebouwd rondom PackML. Het project demonstreert de volledige keten van management‑ en statuslagen tot equipment en control modules, inclusief de typische PLC‑structuur (GVL’s, POUs, DUTs) en een eenvoudige HMI‑laag. 
Functioneel modelleert het een basislijn met een infeed‑conveyor en een lift, waar een product via handshake‑signalen wordt overgedragen. 
Daarmee toont het voorbeeld hoe modules onderling communiceren, hoe sequencing in PackML‑states gebeurt, en hoe alarms, safety‑voorwaarden en statusfeedback door de lagen heen worden afgehandeld. 
Het project is bewust overzichtelijk gehouden zodat het als startpunt kan dienen voor eigen machines of om de GSBF‑/TwinCAT‑modulestructuur te begrijpen, en draait standaard in simulatie om zonder hardware te testen.
